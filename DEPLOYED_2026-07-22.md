## 为什么没有把 chenyme「最新版 v3.0.7」直接盖到 8010

1. 上游自 **v3.0.0** 起是 **Go + React 全量重写**，不是 Python 热升级。
2. 本机 **没有 Go / Docker**，无法编译运行 v3 后端。
3. regkit 依赖 Python 管理 API：
   - `/admin/api/tokens`
   - `/admin/api/tokens/add`
   - `/admin/api/tokens/import/sub2`
4. 若直接换纯净 v3，会 **丢失全部本地改动**，号池对接也会断。

## 正确「升级」定义（已完成）

在 **不丢本地改动、不丢 data/** 的前提下，对齐上游 Python 线最新 `v2.0.4.rc4` 并保持本地 peer/sub2 定制在线。
